# React template

## Installation

```
npm i
```

## Scripts

**HTML Validation**

```
npm run validate-html
```

**JS Validation (JSHint)**

```
npm run jshint
```

**CSS Validation**

```
npm run validate-css
```

**Run DEV mode**

```
npm run dev-mode
```

**Build your bundle**

```
npm run build
```
